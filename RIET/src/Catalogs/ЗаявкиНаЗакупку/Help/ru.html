Справочник "POs"

В справочнике "POs" хранится общая информация по заказанным в системе SWPS товарам или услугам (BORG, Supplier, Purchaser name и т. д.).

Детальная информация (описание, количество, цена и т. д.) по каждому товару или услуге хранится в справочнике "PO lines".

На форме справочника отображается как общая информация, так и детальная.

Данные загружаются в справочник из csv-файла, выгруженного из SWPS, с помощью обработки "PO loading".

Структура

  • No. (строка) - номер заявки на закупку.
    Обязательный для заполнения, если PO не помечена на удаление.
  • For deletion (булево) - признак того, что данная PO предназначена для удаления.
  • BORG (справочник BORGs) - buying organization - код подразделения OFS в SWPS и в Gold.
  • PO date (дата) - дата заявки на закупку.
  • ISUP (строка) - посредник.
  • Supplier (строка) - поставщик.
  • Supplier country (строка) - страна поставщика.
  • Supplier type (строка) - тип поставщика.
  • Purchaser name (строка) - имя заказчика (человек, осуществивший заказ).
    Может понадобится для выяснения технических деталей при таможенном оформлении.
  • Submitter name (строка) - податель.
  • Ship-to label (строка) - грузополучатель.
  • Comment (строка) - комментарий.
