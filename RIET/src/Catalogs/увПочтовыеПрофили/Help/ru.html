Справочник "Почтовые профили"

Справочник хранит почтовые профили, которые используются для установления соединения с почтовыми серверами и содержат информацию об адресате электронных писем.

Имя пользователя – логин для входа на почту, обычно это имя совпадает с частью адреса электронной почты, расположенной слева от знака @.
Пароль – пароль для входа на почту.
Сервер входящей почты (POP3) – сервер для входящих сообщений. Сведения об этом сервере можно получить у поставщика услуг Интернета или у администратора локальной сети.
Сервер исходящей почты (SMTP) – сервер для исходящих сообщений. Сведения об этом сервере можно получить у поставщика услуг Интернета или у администратора локальной сети.
Порт POP3 – порт сервера POP3 (обычно 110).
Порт SMTP – порт сервера SMTP (обычно 25).
Адрес отправителя – адрес электронной почты, который используется для отправки сообщений по данному профилю.
Адрес получателя – адрес электронной почты, на который будет отправлено сообщение содержащее файл выгрузки данных.
Имя пользователя SMTP – логин для входа на сервер SMTP, обычно на этом сервере регистрация не производится, в противном случае нужно установить пометку "SMTP-серверу требуется проверка подлинности".
Пароль SMTP – пароль для входа на сервер SMTP.
