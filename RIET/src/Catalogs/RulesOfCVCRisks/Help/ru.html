Справочник "Rules of CVC risks"

Справочник "Rules of CVC risks" входит в подсистему "CVC risks".
Предназначен для описания правил поиска товаров с высокими рисками корректировки таможенной стоимости.

Существуют 2 типа правил:

  1. Part number = some part number, то есть поиск товаров по точному совпадению парт номера.
  2. Description = *some part of description*, то есть поиск таких товаров, у которых в описании содержится указанная последовательность символов.

Для каждого правила описываются детали (каков предполагаемый размер корректировки, какой код ТН ВЭД и др.).

Для удобства правила могут быть объединены в группы.

Реквизиты шапки

  • Code (строка) - автоматически формируемый код правила.
  • For deletion (булево) - признак того, что это правило будет удалено.
  • Field (перечисление Fields for rules of CVC risks) - поле, по которому будет происходить поиск (Part number или Description). Обязательное для заполнения поле, если правило не помечено на удаление.
  • Value (строка) - искомая последовательность символов.
    Обязательное для заполнения поле, если правило не помечено на удаление.
  • Customs demand (число) - предполагаемый размер корректировки.
    Обязательное для заполнения поле, если правило не помечено на удаление.
  • Currency (справочник Currencies)
    Обязательное для заполнения поле, если правило не помечено на удаление.
  • Goods description (строка)
    Обязательное для заполнения поле, если правило не помечено на удаление.
  • Created by (справочник Users) - пользователь, создавший правило.
    Заполняется автоматически.
  • Creation date (дата) - дата создания правила.
    Заполняется автоматически.
  • Modified by (справочник Users) - пользователь, выполнивший последние интерактивные изменения в правиле.
    Заполняется автоматически.
  • Modification date (дата) - дата последнего интерактивного изменения правила.
    Заполняется автоматически.

Таблица "POAs"

В таблице перечисляются порты, на которые распространяется это правило.
Если не указан ни один порт - правило распространяется на все порты.

  • POA (справочник Sea and air ports) - point of arrival
    Система автоматически удаляет пустые и повторяющиеся строки.
