Контрагенты (Clients)

Поля справочника:

  • Наименование
  • Код 1С
  • ИНН и КПП
  • Полное (юридическое) наименование
  • Адрес (на закладке Контакты)
  • Lawson ID (загружено из Siebel)
  • Контрагент lawson
  • Siebel account
  • Description – английское наименование
  • Ответственный – пользователь ИБ (ссылка) или имя ответственного (строка), загружено из Siebel
  • Проверено – отметка того, что все поля проверены
  • Список договоров – на закладке Счета и договоры

 

Пояснения по заполнению

  • На каждое обособленное подразделение (уникальное сочетание ИНН, КПП, полное наименование) заводится отдельная карточка в справочнике, соответственно, могут быть случаи, когда несколько контрагентов имеют одинаковые Lawson ID
  • Договор привязывается только к одному контрагенту
  • В карточке можно указать головную организацию, для этого нужно установить отметку Входит в холдинг
  • Если контрагент относился к разным инвойсинговым центрам, то его помещали только в одну папку, задвоенных контрагентов быть не должно
  • В программе 1С в интерфейсе Sales book справочник контрагентов доступен из меню Invoicing->Справочники->Контрагенты

Счета и договоры

На данной закладке редактируются банковские счета контрагента и договоры, заключенные предприятием с контрагентом.

Жирным шрифтом отмечаются основной договор и расчетный счет, которые можно изменить соответствующими кнопками командных панелей «Установить основным».
