Справочник "Sub MOTs"

Справочник "Sub MOTs" входит в "подсистему Local distribution". Предназначен для хранения способов перевозки, используемых в локальной транспортировке.

Реквизиты шапки

  • Code (строка) - код
    Обязательный для заполнения, если элемент не помечен на удаление

  • For deletion (булево) - признак того, что данный элемент помечен на удаление

  • MOT (справочник MOTs) - способ перевозки, которому подчинен Sub MOT 
    Обязательный для заполнения реквизит, не может быть помечен на удаление
