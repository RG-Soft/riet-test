Справочник "Contracts"

Справочник "Contracts" входит в подсистему "DOC".
Предназначен для ведения перечня контрактов с продавцами.

Реквизиты шапки

  • Name (строка)
    Обязательный для заполнения, если элемент не помечен на удаление.
  • For deletion (булевое) признак того, что данный элемент помечен на удаление.
  • Seller (справочник Sellers)
    Обязательный для заполнения, если элемент не помечен на удаление.
  • Consign to (справочник Consign to)
    Обязательный для заполнения, если элемент не помечен на удаление.
  • Incoterms (справочник Incoterms)
    Обязательный для заполнения, если элемент не помечен на удаление.
  • Date (дата)
    Обязательный для заполнения, если элемент не помечен на удаление.
  • Expiry date (дата)
    Обязательный для заполнения, если элемент не помечен на удаление.
    Не может быть раньше Date.
