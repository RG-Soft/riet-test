Справочник "Bank accounts for customs payments"

Справочник входит в подсистему "Payments".
Предназначен для ведения перечня банковских счетов, с которых осуществляются расчеты с таможней.

Реквизиты

  • No. (строка)
    Обязательный для заполнения реквизит, если элемент не помечен на удаление.
  • Name (строка)
    Обязательный для заполнения реквизит, если элемент не помечен на удаление.
  • Type of payments (перечисление Types of customs payments)
    Обязательный для заполнения реквизит, если элемент не помечен на удаление.
  • Sold-to (справочник Sold-to) - юридическое лицо.
    Обязательный для заполнения реквизит, если элемент не помечен на удаление.
