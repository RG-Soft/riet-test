Справочник Operating locations (GOLD)

Справочник Operating locations входит в подсистему Import и предназначен для хранения списка операционных локаций.
Операционная локация указывается в импортных инвойсах (кроме TMS инвойсов).
Для каждой операционной локации указан склад по-умолчанию, на который следует доставлять товары.
Таким образом, ведение перечня операционных локаций позволяет автоматически определять место доставки импортных товаров.

Реквизиты шапки

  • Code (строка)
    Обязательный для заполнения реквизит.
    Контролируется уникальность.

  • Name (строка)
    Обязательный для заполнения реквизит.

  • For deletion (булево) - признак того, что данный элемент помечен на удаление.

  • Country (справочник Countries of process levels)
    Обязательный для заполнения реквизит.

  • Address 1, 2, 3, City / location (строки) - адресные линии
    Address 1 - обязательный для заполнения реквизит, так как по нему определяется операционная локация при загрузке GOLD инвойсов.

  • Default WH (справочник Warehouses) - склад, на который по умолчанию будут доставляться товары.
    Обязательный для заполнения реквизит.

  • Created by (справочник Users) - пользователь, создавший операционную локацию.

  • Creation date (дата и время) - дата создания операционной локации.

  • Modified by (справочник Users) - пользователь, внесший последние изменения в операционную локацию.

  • Modification date (дата и время) - дата последнего изменения операционной локации.
