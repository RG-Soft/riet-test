Справочник "Customs regimes"

Справочник входит в подсистему "Customs documents".
Предназначен для ведения перечня таможенных процедур и режимов

Реквизиты справочника

  • No. (строка)
    Обязательный для заполнения, если элемент не помечен на удаление.
    Контролируется уникальность.

  • For deletion (булево) - признак того, что данный элемент помечен на удаление.

  • Import / export (перечисление Import / export)
    Обязательный для заполнения, если элемент не помечен на удаление.
