Справочник "Locations (former Warehouses)"

Справочник "Locations" входит в подсистемы "Import/Export" и "Local distribution".
Предназначен для ведения списка локаций.

Реквизиты справочника

  • Code (строка)
    Обязательный для заполнения, если локация не помечена на удаление.
    Контролируется уникальность.

  • Name (строка)
    Обязательный для заполнения, если локация не помечена на удаление.

  • For deletion (булево) - признак того, что данная локация помечена на удаление.

  • Address (строка)
    Обязательный для заполнения, если локация не помечена на удаление.

  • Warehouse (булево) - признак того, что это склад.
