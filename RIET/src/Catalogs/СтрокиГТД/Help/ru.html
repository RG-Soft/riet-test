Справочник "CCD lines"

Справочник "CCD lines" входит в подсистему "Customs documents".
Справочник предназначен для хранения данных строк ГТД (графы 31-46).
Общая информация по ГТД хранится в документе "CCD".
Строки ГТД в свою очередь делятся на товары строк ГТД, которые хранятся в справочнике "CCD line goods".
Вся ГТД вместе со строками и товарами строк может быть загружена автоматически из xml-файла с помощью обработки "CCD loading". 

Структура справочника

Реквизиты шапки

  • CCD (документ "CCD")
    Обязательное для заполнения поле.
    Нельзя изменить строку ГТД, если документ ГТД проведен.

  • CCD line no. (число)
    Обязательное для заполнения поле.
    Не должно существовать другой строки ГТД с таким же номером.

  • Good description (строка)
  • Gross weight (число)
  • Net weight (число)
  • Customs cost (число)
    Обязательное для заполнения поле.
  • Statistical cost (число)
  • TN VED code (строка)
  • Origin country code (строка)
  • Supplementary qty. (число)
  • Supplementary UOM name (строка)
  • Supplementary UOM code (число)

Табличная часть "Presented documents"

Эта табличная часть - аналог графы 44 (Дополнительная информация. Предоставленные документы. Сертификаты и разрешения) ГТД.

  • Name (строка) - наименование документа
  • Number (строка) - номер документа
  • Date (строка) - дата документа 
  • Presented document mode code (строка) - код вида документа

Табличная часть "Payment calculation"

Эта табличная часть - аналог графы 47 (Исчисление платежей) ГТД.

  • Payment mode code (строка)
  • Payment mode (справочник CCD payment modes)
    Обязательное для заполнения поле.
  • CVC (булево) - признак того, что в строке указана сумма корректировки платежа
  • Payment amount (число)
  • Payment currency code (число)
  • Tax base (число)
  • Tax base currency code (число)
  • Rate (число)
  • Rate type code (строка)
  • Currency use date (строка)
  • Rate use date (строка)
  • СП (строка)
