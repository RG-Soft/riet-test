Справочник "Catalog"

Справочник "Catalog" предназначен для ведения перечня товаров.
Это неполный перечень товаров, а только часть, которая ведется специалистами компании Шлюмберже.

Реквизиты шапки

  • Code (строка) - код.
    Обязательный для заполнения.
    Контролируется уникальность.

  • For deletion (булево) - признак того, что данный продавец будет удален.

  • Description eng. (строка)

  • Description rus. (строка)

  • Fiscal type (справочник Fiscal types) - тип товара, испольуемый для фискальных целей.
