Справочник "Counties / HUBs"

Справочник входит в подсистему Import / Export".
Предназначен для ведения перечня хабов и стран.

Реквизиты шапки

  • Code (строка)
    Обязательный для заполнения, если элемент не помечен на удаление.
    Контролируется уникальность.
  • Name (строка)
    Обязательный для заполнения, если элемент не помечен на удаление.
  • For deletion (булево) - признак того, что данный элемент помечен на удаление.
  • E-mail boxes list (строка) - автоматически формируемый список адресов из таблицы "E-mail boxes".
    Используется для удобного отображении адресов в списке хабов.

Таблица "E-mail boxes"

Таблица "E-mail boxes" используется для хранения перечня почтовых ящиков хабов.

  • E-mail box (строка)
    Обязательный для заполнения, если элемент не помечен на удаление.
