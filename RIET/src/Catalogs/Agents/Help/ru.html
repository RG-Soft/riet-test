Справочник "Agents"

Справочник "Agents" входит в подсистемы "DOC", "Shipment", "Export", "Customs documents",  и "Services".
Предназначен для ведения перечня контрагентов, предоставляющих услуги по таможенному оформлению, транспортировке, хранению товаров и др.

Реквизиты шапки

  • Code (строка)
    Обязательный для заполнения, если элемент не помечен на удаление.
    Контролируется уникальность.

  • Name (строка)
    Обязательный для заполнения, если элемент не помечен на удаление.

  • For deletion (булево) - признак того, что данный элемент помечен на удаление.

  • ИНН (строка)

  • E-mail boxes list (строка) - автоматически формируемый список адресов из таблицы "E-mail boxes".
    Используется для удобного отображении адресов в списке агентов.

Таблица "E-mail boxes"

Таблица "E-mail boxes" используется для хранения перечня почтовых ящиков контрагента с привязкой к методу транспортировки товаров.

  • MOT (справочник Modes of transportation) - mode of transportation
    Обязательный для заполнения, если агент не помечен на удаление.
  • E-mail box (строка)
    Обязательный для заполнения, если агент не помечен на удаление.
