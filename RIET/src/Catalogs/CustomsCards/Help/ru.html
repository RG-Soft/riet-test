Справочник "Customs cards"

Справочник "Customs cards" входит в подсистему Payments и предназначен для ведения перечня таможенных карточек.

Реквизиты шапки

  • Code (строка) - код
    Обязательный для заполнения.
    Контролируется уникальность.

  • Card holder (строка)
    Обязательный для заполнения.

  • CCA (справочник Agents)
    Обязательный для заполнения.

  • For deletion (булево) - признак того, что данный проект удален.

  • Expiry date 
    Обязательный для заполнения.
