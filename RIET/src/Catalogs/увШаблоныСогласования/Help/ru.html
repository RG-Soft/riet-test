Справочник "Шаблоны согласования"

 

Справочник содержит список шаблонов согласования. Справочник доступен из меню "Монитор сопровождения"/"Справочники"/"Шаблоны согласования". В табличной части шаблона в необходимом порядке указывается список лиц, которые будут участвовать в процедуре согласования. 

Используется для определения процедуры согласования документов Заявка на проведение работ. Справочник подчинен справочнику Пользователи и справочнику Группы пользователей. Для каждого пользователя указывается основной шаблон, по которому будет автоматически заполняться список согласования при создании нового документа заявки. Также можно указать некоторый шаблон для группы пользователей и он станет доступен каждому пользователю из этой группы.
