Справочник "Services"

Справочник входит в подсистему "Services".
Предназначен для ведения перечня услуг, оказываемых поставщиками в процессе импорта или экспорта товара.

Справочник является иерархическим, то есть услуги можно группировать в папках для удобства поиска нужной услуги или для целей анализа группы услуг в отчетах.

Реквизиты шапки

  • No. (строка)
    Обязательный для заполнения реквизит.
    Контролируется уникальность.
  • Name (строка) - автоматически формируемое представление услуги как [No.] - [Description].
  • Group (родитель) - группа, в которую входит данная услуга.
  • This is a group (булево) - признак того, что это не услуга, а группа услуг.
  • For deletion (булево) - признак того, что данный элемент помечен на удаление.
  • Description eng. (строка)
    Обязательный для заполнения реквизит
  • Sum calculation method (перечисление Sum calculation methods) - способ расчета стоимости.
    Обязательный для заполнения реквизит. Возможные значения:
    • Standard tariff
    • Cost plus
    • Quoted
  • Allocation area (перечисление Allocation areas) - область товаров, на которую распределяется стоимость данной услуги (обязательный для заполнения реквизит.
    Обязательный для заполнения реквизит. Возможные значения:
    • Shipment - стоимость сервиса распределяется на все товары, относящиеся к указанной в шапке Поставке
    • CCD - стоимость сервиса распределяется на все товары, относящиеся к указанной ГТД
    • Goods - стоимость сервиса распределяется на определенный состав товаров, указанный в табличной части  "Goods"
  • Allocation method (перечисление Allocation methods) - пропорция распределения.
    Обязательный для заполнения реквизит. Возможные значения:
    • Equally
    • By chargeable weight
    • By sum
  • Duties recharge (булево) - признак того, что данная услуга оплачивается брокером, но перевыставляется компании Шлюмберже.
    В этом случае область распределения стоимости услуги может быть только на CCD.

Табличная часть "POAs"

В табличной части перечисляются порты, для которых действует эта услуга.
Если табличная часть пустая - услуга действует на всех портах.

  • POA (справочник Sea and Air ports)
    Система автоматически удаляет повторяющиеся и пустые строки.
