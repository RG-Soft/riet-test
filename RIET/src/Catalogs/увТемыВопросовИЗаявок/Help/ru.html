Справочник "Темы вопросов и заявок"

 

Справочник предназначен для хранения тем для документов "Вопрос" и "Заявка на выполнение работ". Информация о темах используется как аналитический разрез при анализе проблем пользователей. Справочник можно использовать для группировки документов (вопросов и заявок) по возникающим проблемам или для выделения отдельных блоков внедрения.

Для строгой классификации тем используется реквизит "Вид темы".

Справочник имеет иерархическую структуру для упорядочения тем по группам.

Справочник доступен из меню "Монитор сопровождения"/"Справочники"/"Темы вопросов и заявок".
