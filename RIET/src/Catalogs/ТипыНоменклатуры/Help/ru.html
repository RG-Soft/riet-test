Справочник "Fiscal types"

Справочник предназначен для ведения перечня фискальных типов товаров.

Реквизиты

  • Code (строка)
    Обязательный для заполнения.
    Контролируется уникальность.
  • Name (строка)
    Обязательный для заполнения.

  • For deletion (булево) - признак того, что элемент помечен на удаление.
