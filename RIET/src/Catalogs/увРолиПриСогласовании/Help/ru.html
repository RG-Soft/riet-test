Справочник "Роли при согласовании"

 

Справочник содержит список ролей пользователей в процедуре согласования. Используется для составления шаблонов согласования. Справочник содержит предопределенный элемент "Согласовал". Справочник доступен из меню "Монитор сопровождения"/"Справочники"/"Роли при согласовании".

Подсистема  "Монитор сопровождения" ред 2.0
