Справочник "Consign to"

Справочник "Contracts" входит в подсистему "DOC", "Export" и "Services".
Предназначен для ведения перечня юридических лиц, от имени которых может производиться таможенное оформление товаров.

Реквизиты шапки

  • Code (строка)
    Обязательный для заполнения, если элемент не помечен на удаление.
    Контролируется уникальность.
  • Name (строка)
    Обязательный для заполнения, если элемент не помечен на удаление.
  • For deletion (булевое) - признак того, что данный элемент помечен на удаление.
  • Address 1, 2, 3, 4, 5 (строки)
    Обязательный для заполнения, если элемент не помечен на удаление.
  • E-mail boxes list (строка) - автоматически формируемый список всех почтовых адресов

Таблица "E-mail boxes"

Таблица "E-mail boxes" используется для хранения перечня почтовых ящиков.

  • E-mail box (строка)
    Обязательный для заполнения, если элемент не помечен на удаление.
