Справочник "Sellers"

Справочник "Sellers" входит в подсистему DOC и предназначен для ведения перечня продавцов.

Реквизиты шапки

  • Code (строка) - код.
    Обязательный для заполнения.
    Контролируется уникальность.

  • Name (строка) — имя.
    Обязательный для заполнения.

  • For deletion (булево) - признак того, что данный продавец будет удален.
