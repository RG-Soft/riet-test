Справочник "PSA contracts"

Справочник "PSA contracts" входит в подсистему Import и предназначен для ведения перечня проектов.
От проекта могут зависеть специальные требования к импорту/экспорту товаров (например, гарантийное обязательство Сахалинской Энергии при ввозе без уплаты таможенных пошлин).

Реквизиты шапки

  • Code (строка) - код
    Обязательный для заполнения.
    Контролируется уникальность.

  • Name (строка) — имя
    Обязательный для заполнения.

  • For deletion (булево) - признак того, что данный проект удален.

  • Segment (справочник Segments)

  • Expiry date - обязательный для заполнения.

  • Country (справочник Countries (RCA)) - обязательный для заполнения.

  • Client (строка) - обязательный для заполнения.

  • Zero customs charges (булево) - признак того, что по данному проекту не обязательно указывать платежи.
