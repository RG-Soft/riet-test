Справочник "События с основными средствами"

Справочник предназначен для хранения перечня событий с основными средствами организации, например, принятие к учету ОС, перемещение ОС, модернизация ОС, ввод в экусплуатацию ОС и т.п.

Для каждого элемента справочника необходимо выбрать соответствующий вид события с основными средствами. Список возможных видов событий с основными средствами задан в программе в перечислении "Виды событий с основными средствами".

При заполнении в программе документов по учету основных средств в поле "Событие" выбирается соответствующий элемент справочника "События с основными средствами". При проведении таких документов формируются записи в регистре сведений "События ОС организаций".
