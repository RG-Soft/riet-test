Справочник "Sea and air ports"

Справочник "Sea and air ports" входит в подсистемы "DOC", "Shipment", "CVC risks", "Export", "Local distribution" и "Services".
Предназначен для ведения перечня портов.

Реквизиты шапки

  • Code (строка) - код.
    Обязательный для заполнения, если порт не помечен на удаление.
    Контролируется уникальность.

  • Name (строка) — имя.
    Обязательный для заполнения, если порт не помечен на удаление

  • For deletion (булево) - признак того, что данный порт помечен на удаление

  • Group (родитель справочника) - группа, в которую входит порт

  • Sakhalin (булево) - признак того, что данные порт находится на Сахалине
    Значение признака должно совпадать со значением признака в Group

  • Default warehouse (справочник Warehouses) - склад, используемый по умолчанию для хранения товаров, вывезенных из данного порта
    Обязательный для заполнения, если порт не помечен на удаление
