Справочник "User PODs EMail Tempates"

Справочник "User PODs EMail Templates" входит в подсистему "DOC".
Предназначен для хранения списка шаблонов электронных писем пользователя в разрезе хабов.
Эти шаблоны потом используются для автоматизированной отправки писем о переводе DOC с этапа на этап прямо из информационной системы.

Реквизиты шапки

  • Code (строка)
    Обязательный для заполнения, если элемент не помечен на удаление.
    Контролируется уникальность.

  • For deletion (булево) - признак того, что данный элемент помечен на удаление.

  • User (справочник Users)
    Обязательный для заполнения, если элемент не помечен на удаление.

  • POD (справочник HUB Countries)
    Обязательный для заполнения, если элемент не помечен на удаление.

  • Consign to (справочник Consign to)
    Обязательный для заполнения, если элемент не помечен на удаление.

  • Subject (строка)

  • Body (строка)
