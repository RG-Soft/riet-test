Справочник "Parent companies (RCA)"

Справочник "Parent companies (RCA)" входит в подсистемы "Import / Export", "Customs documents batches" и "Local distribution".
Предназначен для ведения укрупненного перечня юридических лиц, от имени которых работает компания Шлюмберже.

Реквизиты шапки

  • Code (строка)
    Обязательный для заполнения реквизит.
    Контролируется уникальность.
  • Name (строка)
    Обязательный для заполнения реквизит.
  • For deletion (булево) - признак того, что данный элемент помечен на удаление.
  • Start of export to TMS (дата) - дата начала выгрузки заявок на экспорт в TMS.
  • Name for Move it (строка) - имя юридического лица в информационной системе Move it. 
  • Lawson (булево) - призна того, что данные этого юридического лица ведутся в информационной системе Lawson.
  • Company no. (строка) - номер компании в Lawson.
    Обязательный для заполнения для Lawson компаний.
  • Country (справочник Countries (RCA)) - обязательный для заполнения.
  • Address 1 (строка) - обязательный для заполнения.
  • Parent co. for payments (справочник Parent companies (RCA))
    Обязательный для заполнения реквизит, если не указан, то устанавливается сам элемент справочника.
