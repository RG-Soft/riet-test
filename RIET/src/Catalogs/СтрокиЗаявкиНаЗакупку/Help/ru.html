Справочник "PO lines"

Справочник "PO lines" входит в подсистему "PO" и предназначен для хранения детальной информации по заказанным в системе SWPS товарам или услугам (описание, количество, цена и т. д.).

Общая для нескольких PO lines информация (BORG, Supplier, Purchaser name и т. д.) хранится в справочнике "POs".

На форме справочника отображается как общая информация, так и детальная.

Данные загружаются в справочник из csv-файла, выгруженного из SWPS, с помощью обработки "PO loading".

Структура

  • PO (справочник POs) - общие данные заявки на закупку.
    Обязательный для заполнения, если PO line не помечена на удаление.
  • Line no. (число) - номер строки заявки на закупку.
    Обязательный для заполнения поле, если PO line не помечена на удаление.
  • No. (строка) - уникальный код, используется как представление PO line в списках.
    Формируется автоматически как PO no. + Line no.
  • For deletion (булево) - признак того, что данная PO line предназначена для удаления.
  • Part description (строка) - описание заказываемого товара или услуги.
  • Name (строка) - первые 150 символов поля Part description. Служебное поле, необходимое для быстрого поиска.
  • Supplier P/N (строка) - код заказываемого товара или услуги.
  • Catalog (справочник Catalog) - ссылка на каталог, если заказываемый товар содержится в каталоге компании Шлюмберже.
  • Quantity (число) - количество.
  • UOM (справочник UOMs) - единица измерения.
  • Unit cost (число) - цена.
  • Currency (справочник Валюты) - валюта.
  • Extended cost (число) - сумма в указанной валюте.
  • Extended cost (usd.) (число) - сумма в долларах.
  • ERP treatment (перечисление ERP treatments) - управленческая классификация затрат.
  • AU (справочник AUs) - кост-центр.
    Обязательное для заполнение поле для Lawson компаний.
  • Activity сode (строка)
  • Type of requisition (строка) - тип запроса.
  • Commodity (строка) - назначение расходов.
  • Comodity type (строка) - тип назначения расходов.
  • Ship method (строка) - способ перевозки.
  • Shipping instructions (строка) - инструкции по транспортировке.
  • Special instructions (строка) - специальные инструкции.
