Справочник "Структурные подразделения"

Справочник предназначен для дополнительного анализа данных по подразделениям компании.

Информация о подразделениях компании заполняется в документы, в дальнейшем в отчетах можно проводить анализ по подразделениям компании.

Справочник вызывается для ввода и редактирования информации из пункта меню "Монитор сопровождения" / "Справочники" /"Подразделения".

Добавление и редактирование информации производится в форме списка.
Для справочника реализована иерархическая структура, с помощью которой можно задавать структуру подразделений, которая имеется в компании.
