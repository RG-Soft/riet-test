Справочник "Accounting units"

Справочник "Accounting units" (AUs) предназначен для хранения списка используемых кост-центров.

Кост-центры загружаются в базу автоматически. При добавлении кост-центра вручную требуется также добавить его в регистр сведений "Сегменты кост-центров".

Реквизиты шапки

  • No(строка) - код.

  • Name (строка) — имя.

  • Segment (справочник Segments) - сегмент, группа на верхнем уровне.

  • Sub segment (справочник Segments) - субсегмент, группа, подчиненная группе Segment.

  • Sub sub segment (справочник Segments) - субсубсегмент, элемент, подчиненный группе Sub Segment 

  • Tax registration (справочник "Подразделения организации") - налоговая регистрация (группа).

  • Подразделение организации (справочник "Подразделения организации") - подразделение.

  • Default activity (строка) - активити, используемое по-умолчанию для данного кост-центра.

  • OnlyMS (булево) - признак того, что кост-центр может быть использован только для материалов и не может быть использован для основных средств.

  • NonLawson (булево) - признак того, что это кост-центр, соответствия которому нет в Lawson.
