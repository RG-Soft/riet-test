Справочник "Customs payments kinds"

Справочник "Customs posts" входит в подсистемы "Customs documents".

Предназначен для ведения перечня видов таможенных платежей.

Реквизиты

  • No. (строка)
    Обязательный для заполнения, если элемент не помечен на удаление.

  • Name (строка)
    Обязательный для заполнения, если элемент не помечен на удаление.

  • For deletion (булево) - признак того, что данный элемент помечен на удаление.

  • Type (перечисление Customs payments types) - тип платежа (более общая классификация по сравнению с видами).
