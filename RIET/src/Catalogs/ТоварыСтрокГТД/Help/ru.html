Справочник "CCD lines goods"

Справочник входит в подсистему "Customs documents".
Справочник предназначен для хранения товаров строк ГТД (графа 31 (Грузовые места и описание товаров)).
Товары с одним кодом ТН ВЭД объединяются в строки ГТД, которые хранятся в справочнике "CCD lines".
Общая информация ГТД хранится в документе "CCD".

Вся ГТД вместе со строками и товарами строк может быть загружена автоматически из xml-файла с помощью обработки "CCD loading".

Нельзя изменить данные товаров строк ГТД, если они относятся к проведенному в системе документу "Invoice lines matching"

Структура справочника

  • CCD line (справочник CCD lines)
    Обязательный для заполнения реквизит.
  • No. (число) - номер товара в строке ГТД
    Обязательный для заполнения реквизит.
    Контролируется уникальность в пределах строки ГТД.
  • Name (строка) - автоматически формируемое представление товара как [No.] - [Characteristics]
  • Сharacteristics (строка) - описание товара
  • Manufacturer (строка)
  • Trademark (строка)
  • Goodsmark (строка)
  • Goodsmarking (строка)
    Система автоматически заменяет русские буквы на латинские, чтобы потом иметь возможность автоматического сопоставления со строками инвойса.
  • Qty (число)
  • UOM name (строка)
  • UOM code (число)
