Справочник "BORGs"

Структура

  • Code (строка) - код.
    Обязательный для заполнения, если BORG не помечен на удаление.
  • Name (строка) - имя.
    Обязательный для заполнения, если BORG не помечен на удаление.
  • For deletion (булево) - признак того, что данный BORG предназначен для удаления.
  • Company (справочник Sold-to) - юридическое лицо.
    Обязательный для заполнения, если BORG не помечен на удаление.
    Не может быть помечен на удаление.
  • Default AU (справочник AUs) - кост-центр, используемый по-умолчанию.
    Обязательный для заполнения, если BORG не помечен на удаление.
    Не может быть помечен на удаление.
    Кост-центр должен быть связан с этим BORG в регистре "AUs and BORGs".
  • Segment (справочник Segments) 
    Обязательный для заполнения, если BORG не помечен на удаление.
    Segment Default AU должен совпадать с Segment.
