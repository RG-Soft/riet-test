Справочник "Modes of transportation"

Справочник "Modes of transportation" входит в подсистемы "DOC", "Shipment", "Export" и "Local distribution".
Предназначен для ведения перечня способов перевозки.

Реквизиты шапки

  • Code (строка) - код
    Обязательный для заполнения, если элемент не помечен на удаление.
    Контролируется уникальность.
  • For deletion (булево) - признак того, что данный элемент помечен на удаление.
  • Target days (число) - целевое время таможенного оформления товаров для отчета Leg 6: on time clearance.
    Обязательный для заполнения, если элемент не помечен на удаление.
