Справочник "Leg7 Warehouses"

Справочник "Leg7 Warehouses" входит в подсистему "Local distribution".
Предназначен для ведения списка складов, с которыми производится обмен данными по локальной транспортировке.

Реквизиты справочника

  • Name (строка) - название склада.
  • For deletion (булево) - признак того, что данный склад помечен на удаление.

  • Location (справочник Locations)
    Обязательный для заполнения, если склад не помечен на удаление.

  • Start of Int. inbound reports (дата)

  • E-mail box (строка)
    Обязательный для заполнения, если заполнена дата Start of Int. inbound reports.

  • POA's list (строка) - список POA строкой.

Табличные части

Points of Arrival - в справочнике должен быть выбран хотя бы один Point of Arrival.
