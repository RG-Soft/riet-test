Справочник UOMs

Справочник предназначен для ведения перечня единиц измерений и коэффициентов пересчета в базовые единицы измерения.

Реквизиты шапки

  • Code (строка) - двух или трех буквенное сокращение единицы измерения
    Обязательный для заполнения, если единица измерения не помечена на удаление.
  • Name (строка) - полное название единицы измерения
  • Standard (булево) - признак того, что единица измерения является стандартной.
    Если флажек отмечен, то поля "UOM standard" и "Conversion factor" становятся недоступным, при записи единицы измерения "Conversion factor" становится равным 1, а в реквизит "UOM standard" автоматически подставляется текущая единица измерения.
    При снятии флажка проверяется, что на эту единицу измерения не ссылаются другие единицы измерения.
  • UOM standard (справочник UOMs) - единица измерения в которую будет пересчитываться текущая единица измерения.
    Обязательный для заполнения, если единица измерения не помечена на удаление.
    Должна быть указана стандартная единица измерения (при выборе система показывает только стандартные единицы измерения).
    Не может быть помечена на удаление.
  • Conversion factor (число) - коэффициент пересчета текущей единицы измерения в стандартную.
    Например, значение 2 означает, что одна текущая единица измерения равна двум стандартным единицам измерения.
    Обязательный для заполнения, если единица измерения не помечена на удаление.
    Для стандартной единицы измерения система автоматически подставляет 1.
