Отчет "Вопросы по вопросам"

 

В интерфейсе подсистемы «Монитор сопровождения» отчет «Отчет по вопросам» вызывается по опции основного меню «Монитор сопровождения», пункт «Отчет по вопросам».

Этот отчет разработан с использованием  схемы компоновки данных, что обеспечивает возможность опытному пользователю программы «1С: Предприятие» делать различные настройки для формирования новых форм отчетов с помощью конструктора настроек. 

С помощью стандартной формы настройки формируются диаграммы распределения вопросов по месяцам и количества вопросов по сотрудникам-адресатам.

С помощью стандартной формы настройки также формируется отчет по вопросам в виде таблицы, с предоставлением информации по вопросам в виде конференции.

.

По кнопке «Конструктор настройки» вызывается  форма работы с настройками отчетов.

На правой панели этого окна расположены различные функциональные кнопки.

По кнопке «Стандартная настройка» можно восстановить стандартные настройки отчета. Это  кнопка находится  на правой боковой панели верхнего окна настройки отчета, третья снизу           У кнопки есть всплывающая подсказка.
