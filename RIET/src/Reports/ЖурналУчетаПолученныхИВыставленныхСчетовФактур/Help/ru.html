Отчет предназначен для формирования списка счетов-фактур, исправленных счетов-фактур, корректировочных счетов-фактур, полученных от поставщиков и переданных покупателям. Используется с 01.01.2012 года.

Отчет формируется в соответствии с правилами ведения журнала учета полученных и выставленных счетов-фактур.

Для изменения параметров формирования отчета выполняется по кнопке Настройка. Для изменения параметров формирования отчета нужно установить переключатель в значение Использовать дополнительные настройки.

  • Отбирать по контрагенту - отражаются счета-фактуры по выбранному контрагенту. Если контрагент не указан, то будут выводиться все полученные и выставленные за период счета-фактуры.
  • Группировать по контрагентам - реестр счетов-фактур группируется по контрагентам.
