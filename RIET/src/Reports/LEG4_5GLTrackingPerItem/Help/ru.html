Отчет "GL-tracking"

Отчет входит в подсистему "DOC" и предназначен для мониторинга и анализа нахождения документов "DOC" в различных статусах процесса Green light.

Отчет выводит перечень документов DOC и их реквизиты, а также следующие показатели:

  • SLB aging, days - количество дней, прошедшее с даты DOC до даты Granted (или до текущей даты, если дата Granted не заполнена)
  • Total GL aging, days - количество дней, прошедшее с даты DOC до даты Booked (или до текущей даты, если дата Booked не заполнена)
  • GL aging, days - количество дней, прошедшее с даты Requested до даты Received (или до текущей даты, если дата Received не заполнена)
  • Pending CCA, days - количество дней, которое DOC находится в статусе Pending CCA (имеет смысл только, когда DOC находится в статусе Pending CCA)
  • Pending SLB, days - количество дней, которое DOC находится в статусе Pending SLB (имеет смысл только, когда DOC находится в статусе Pending SLB)
  • Received not Granted, days - количество дней, прошедшее с даты Received до даты Granted (или до текущей даты, если дата Granted не заполнена)
  • Granted not Shipped, days - количество дней, прошедшее с даты Granted до даты Booked (или до текущей даты, если дата Booked не заполнена)

Разные варианты отчета предназначены для анализа разных статусов процесса Green light.
