Отчет "Регистр информации о движении приобретенных товаров"

Регистр формируется для обобщения информации о наличии и движении приобретенных товаров.

Имеет два режима формирования:

  • по товарам на складах;
  • по товарам отгруженным.

Формируется по данным налогового учета товаров (товаров отгруженных).

 

Состав показателей регистра:

 

  • «По товару» - наименование единицы учета товаров.
  • «По партии» - реквизиты документа, по которому была принята к учету данная партия товаров (если предусмотрено ведение партионного учета)
  • «Дата операции» – соответствует дате документа, отражающего расходы, связанные с приобретением товара, или дате списания товара. Значение данного показателя — дата соответствующей проводки по налоговому учету товаров. При отражении в регистре наличия товаров, по которым в отчетном периоде не производились никакие операции, показатель принимает значение даты начала формирования регистра.
  • «Основание операции» – реквизиты документа, на основании которого производилась приемка, списание или возврат товаров.
  • «Сальдо на начало периода (кол.)» – остаток единицы учета товаров на начало периода (в натуральных измерителях) по данным счета налогового учета.
  • «Сальдо на начало периода (сумма)» – остаток единицы учета товаров на начало периода (в стоимостных измерителях) по данным счета налогового учета.
  • «Приход (в натуральных измерителях)» – количество поступивших товаров.
  • «Приход (количество)» – стоимость поступивших товаров.
  • «Возвраты поставщикам (количество)» – количество возвращенного поставщикам товара.
  • «Возвраты поставщикам (сумма)» – стоимость возвращенного поставщикам товара.
  • «Выбытие (количество)» – количество списываемых по первичному документу товаров.
  • «Цена по операции» – зависит от учетной политики организации для целей налогового учета.
  • «Выбытие (сумма)» – стоимость списываемых товаров. Показатель определяется как произведение показателей «Списание (в натуральных измерителях)» и «Цена по операции».
  • «Текущий остаток (количество)» – определяется как сумма остатка на начало проведения операции плюс приход минус возвраты поставщикам, минус списание (в натуральных измерителях).
