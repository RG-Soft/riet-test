Отчет "Import lead time"

Отчет входит в подсистему Import / Export.
Предназначен для отображения средних показателей времени (в днях) по импортным поставкам.

L4: Green light, days считается так:

  • когда в DOC заполнена Granted, то считается количество дней с Даты DOC до Granted;
  • когда не заполнена Granted, заполнена Requested и Requested меньше ATD из Import Shipment, то считается количество дней с Requested до ATD из Import Shipment;
  • когда не подходят первые два условия, считается 0.

L5 A: Consolidation, days считается так:

  • когда в DOC заполнена Granted и Granted меньше ATD из Import Shipment, то считается количество дней с Granted до ATD из Import Shipment;
  • иначе считается 0.

L5 B: In transit, days считается так:

  • считается количество дней от ATD до ATA из Import Shipment.

L6: Customs clearance, days считается так:

  • считается количество дней от ATA до Completed из Import Shipment.
