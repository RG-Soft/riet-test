Регистр сведений "PO tracking subscriptions"

Регистр хранит информацию о том на обновление статусов каких PO / товаров из этих PO подписались пользователи.

Измерения

  • User (справочник Users)
    Обязательный для заполнения реквизит.
  • PO no. (строка)
    Обязательный для заполнения реквизит.
  • PO line no. (число)
