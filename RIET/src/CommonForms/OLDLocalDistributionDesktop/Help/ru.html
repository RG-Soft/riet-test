Local distribution desktop

Рабочий стол локальной транспортировки входит в подсистему Local distribution
Предназначен для удобной консолидации грузовых мест в документы Trips, а также для управления состоянием документов Trips.

Для осуществления этих функций предназначены 2 вкладки: Parcels и Trips.

Вкладка Parcels

Вкладка Parcels предназначена для удобной консолидации грузовых мест в документы Trips.

В верхней части вкладки отображается перечень фильтров, накладываемых на спиок отображаемых грузовых мест.
По-умолчанию, система уже предоставляет некоторый набор фильтров, таких как: Warehouse from, Warehouse to, Urgency и т. д.
Более того, некоторые фильтры система включает при открытии рабочего стола.
Например, включаются фильтры отсекающие грузовые места:

  • еще не доставленные на склад (Collected from port не заполнено)
  • уже включенные в документы Trips (Trip заполнено)
  • переупакованные грузовые места (Repacked равно ложь)
  • отмененные грузовые места (системный фильтр, который невозможно снять)

Впрочем, пользователь может самостоятельно включить или отключить фильтры и даже добавить собственные фильтры, щелкнув правой кнопкой мыши на поле фильтров.

Ниже располагается панель действий, которая позволяет среди прочего:

  • создать новый Parcel
  • переместить выделенные грузовые места в новый документ Trip (кнопка Move to new trip)
  • переместить выделенные грузовые места в существующий документ Trip (кнопка Move to existing trip)
  • удалить выделенные грузовые места из документов Trip (кнопка Remove from trips).

Наконец, нижнее поле отображает список грузовых мест с учетом наложенных фильтров.
Пользователь может выделять несколько грузовых мест с помощью кнопки мыши и зажатой клавиши Ctrl или Shift.
В подвале система отображает объем и вес выделенных грузовых мест.
К выделенным грузовым местам можно применять действия, описанные в предыдущем абзаце.

Некоторые грузовые места выделяются цветом:

  • бледно зеленым выделяются грузовые места, которые уже включены в документы Trip, но еще не выгружены на склад
  • насыщенно зеленым выделяютя грузовые места, которые уже включены в документы Trip и уже выгружены на склад

Вкладка Trips

Вкладка Trips предназначена для управления документами Trips.

Также как и на вкладке Parcels, в верхней части вкладки Trips отображается перечень фильтров, накладываемых на спиок отображаемых документов.
Некоторые фильтры добавлены системой. Другие может добавить пользователь, щелкнув, правой кнопкой мыши на поле фильтров.
Часть фильтров система включает при открытии рабочего стола. Например, включаются фильтры отсекающие документы:

  • уже выгруженные в Move it (Sent to move it не заполнено)
  • отмененные документы (For deletion равно нет)

Ниже располагается панель действий, которая позволяет создать новый Trip вручную, пометить на удаление существующий Trip и т. д.

Наконец, нижнее поле отображает список документов Trip с учетом наложенных фильтров.

Некоторые документы выделяются цветом:

  • бледно зеленым выделяются документы, которые уже выгружены на склад
  • насыщенно зеленым выделяютя документы, которые уже выгружены в Move it
