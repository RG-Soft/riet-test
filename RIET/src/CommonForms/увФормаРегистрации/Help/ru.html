Форма настройки Монитора сопровождения

В форме вводятся данные для регистрации и настройки подсистемы "Монитор сопровождения". Форма доступна из меню "Настройка" обработки "Вопросы и заявки".

На первой странице необходимо указать Регистрационный номер продукта и Лицензионный ключ.

На второй странице указывается префикс для документов подсистемы. Префикс используется для идентификации Мониторов в сервере Мониторов сопровождения, поэтому при использовании сервера необходимо следить за уникальностью префиксов в различных информационных базах. В случае, если изменяется префикс и в информационной базе уже заведены документы подсистемы, после подтверждения пользователя производится перенумерация документов с установкой нового префикса. Для корректного взаимодействия с сервером мониторов сопровождения операцию перенумерации следует повторить на сервере.

На третье странице указывается единица оценки работ. В этих единицах будут оцениваться работы по заявкам. Единицей может являться элемент справочника "Валюты" или строка (такой тип используется, если работы оцениваются в часах, в этом случае в качестве единицы работ следует написать "часов").

На последней странице указываются параметры обмена данными через почтовые серверы. Здесь следует указать почтовый профиль для отправки срочных вопросов. По адресу, указанному в поле "Адрес получателя", будут отправляться почтовые сообщения с содержанием вопроса. Этот профиль является общим для всех пользователей. Также на этой странице можно указать пароль для шифрования файлов выгрузки данных монитора.
