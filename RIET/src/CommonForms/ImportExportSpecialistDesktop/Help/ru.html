Import / export specialist desktop

Рабочий стол специалиста предназначен для оперативного управления процессами импорта/экспорта товаров.
На рабочем столе выводятся:

  • документы DOC, которые еще не вошли в поставку (не заполнена дата Booked). 
  • не закрытые документы Import Shipment.
    Import shipment считается закрытым, когда:
    • для RU, KZ и UZ, когда заполнена дата Collected from port;
    • для AZ и TM, когда заполнена дата Cleared;
    • кроме того, для Process levels с признаком "Leave documents in desktop until all items are in customs files" документы Import Shipment не уходят с рабочего стола пока все товары не войдут в декларации;
  • документы Export requests, у которых не заполнена дата International ATA.
  • отчет LEG 6: On time clearance (см. соответствующий раздел справки).

На каждую вкладку выведены наиболее часто используемые фильтры, но пользователь может добавить дополнительные фильтры через стандартную настройку таблиц.

В подвале отображается итог по выделенным строкам.

Цветовая индикация строк описана в справке к документам DOC, Shipment и Export request.
