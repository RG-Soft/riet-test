Подсистема "Import / export tracking"

Назначение

Подсистема предназначена для мониторинга состояния импортируемых / экспортируемых товаров внешними пользователями.

Структура

Для того, чтобы воспользоваться функциями подсистемы внешний пользователь проходит регистрацию с помощью специальной формы "Registration".

Далее пользователь входит в подсистему, используя LDAP авторизацию.
Мониторинг состояния товаров осуществляется на форме "Import / export tracking".
Здесь также можно оформить подписку на обновление статусов товаров и просмотреть прикрепленные файлы.

Связь с другими подсистемами

Данные для подсистемы предоставляются подсистемами PO, DOC, Shipment и Export.
