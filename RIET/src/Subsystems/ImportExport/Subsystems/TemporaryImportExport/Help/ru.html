Подсистема 'Temporary import / export'

Подсистема Temporary import / export предназначена для контроля сроков нахождения товаров во временном импорте или экспорте.

Товары могут попасть в систему 2мя способами:

  • автоматически при проведении документов ILM или Customs file (light)
    При этом система контролирует, что товары, помеченные как Temporary, сопоставлены только с декларациями с временным режимом.
  • вручную с помощью документа Temporary imp. / exp. transaction

Перечень товаров, находящихся во временном импорте или экспорте, отображается на рабочем столе Temporary imp. / exp. desktop.

Кроме того, система периодически присылает уведомления о товарах с истекающим сроком на адреса электронной почты, указанные в справочнике Process levels.
Система присылает уведомления один раз в неделю по понедельникам.
В письмо включаются те товары, срок временного импорта / экспорта которых истекает в течение 3х месяцев.

Из рабочего стола создаются документы Temporary imp. / exp. transaction, которые позволяют отразить пролонгацию, первод в свободное обращение и т. д.
Однако для того, чтобы поставить товары на временный контроль вручную, этот документ необходимо создавать не из рабочего стола, а из списка документов.
