Подсистема "Import / Export logistics"

Назначение

Подсистема "Import / Export logistics" предназначена для автоматизации логистических процессов импорта и экспорта товаров.

Структура подсистемы

Подсистема "Import / Export logistics" состоит из более мелких подсистем:

  • PO
  • Import
  • Export
  • Temporary import / export
  • Import / export tracking

Связь с другими подсистемами

Для отражения деклараций, учета таможенных платежей, определения фискальной стоимости грузов и выгрузки данных в Lawson предназначена отдельная подсистема "Customs".

Кроме того, стоимость грузов может включать стоимость услуг, оказанных другими контрагентами (например, услуги брокера).
Стоимость этих услуг учитывается в подсистеме "Services".

Наконец, сведения о текущем состоянии грузов, пунктах отправления и прибытия используются в подсистеме "Local distribution".
