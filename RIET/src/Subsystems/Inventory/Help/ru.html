Подсистема "Inventory"

Подсистема "Inventory" предназначена для количественного и фискального учета inventory-товаров.

Источником данных для подсистемы служат:

  • Lawson отчет xr220, в котором содержатся количественные и управленческие данные, но не содержится фискальных данных
  • Подсистема "Tengible assets", в которой накапливаются фискальные данные для принятия в подсистему "Inventory"

Количественные данные из отчета xr220 загружаются с помощью обработки Загрузка из Lawson.
Эта обработка создает документы Поступление, Перемещение, Списание и Корректировка, в которых заполнены только количественные данные, но не заполнены фискальные суммы.

Фискальные данные могут быть заполнены вручную, или полуавтоматически из подсистемы "Tengible assets".

После заполнения фискальных данных можно рассчтитать себестоимость товаров.
Себестоимость рассчитывается с помощью партионного учета по методу FIFO.
Для расчета себестоимости или восстановления хронологической последовательности партионного учета предназначена обработка Восстановление партионного учета.

Сводные данные о количестве и себестоимости товаров можно посмотреть в отчете Ведомость по партиям.
