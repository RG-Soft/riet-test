Документ "Customs deposit refund to bank account"

Документ "Customs deposit refund to bank account" входит в подсистему "Payments".
Предназначен для отражения возврата суммы депозита (полной или частичной) по таможенной расписке на банковский счет.

Структура документа

  • No. (строка)
    Обязательный для заполнения реквизит, заполняется автоматически. 
  • Date (дата)
    Обязательный для заполнения реквизит.
  • For deletion (булево) - признак того, что документ помечен на удаление.
  • Posted (булево) - признак того, что документ проведен.
  • Customs bond (документ Customs files (light)) - с типом транзакции Customs bond closing (решение по таможенной расписке)
    Обязательный для заполнения реквизит. 
  • Sum (число) 
    Обязательный для заполнения реквизит. 
    При проведении документа проверяется, что сумма не превышает сумму к возврату, указанную в Customs bond closing. 
  • Description (строка) - описание платежа.
  • Creation Date (дата) - дата создания документа.
  • Created by (справочник Users) - заполняется автоматически пользователем, который создал документ.
  • Modification date (дата и время) - заполняется автоматически датой последнего интерактивного изменения документа. 
  • Modified by - (справочник Users) - заполняется автоматически пользователем, который внес последние изменения в документ.
