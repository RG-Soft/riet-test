Документ "Trip"

Документ "Trip" входит в подсистему "Local distribution".
Предназначен для отражения локальной транспортировки грузовых мест.

Консолидация грузовых мест для локальной траспортировки происходит в специальной форме "Local distribution desktop".

В новом документе указывается способ перевозки, планируемая и фактическая дата отправления, список грузовых мест, а также планируемые даты прибытия в конечные точки маршрута.

Далее информация по грузовым местам отправляется на склад.

После подтверждения склада документ проводится и данные выгружаются в систему "Move it". Нельзя выгрузить помеченный на удаление Trip.

Реквизиты шапки

  • No (строка) - номер документа, присваивается системой автоматически.
  • For deletion (булево) - признак того, что документ помечен на удаление.
  • Warehouse from (справочник Destination points)
    Обязательный для заполнения реквизит.
    Warehouse from не должен быть помечен на удаление.
  • Sub MOT (справочник Sub MOTs) - mode of transportation.
    Обязательный для заполнения реквизит.
    Sub MOT не должен быть помечен на удаление.
  • ETD (дата) - estimated time of departure.
    Обязательный для заполнения реквизит.
  • ATD (дата) - actual time of departure.
    Обязательный для заполнения реквизит.
    Не может быть позже текущей даты.
    Не может быть позже "Sent to Move it".
  • Waybill (строка)
    Обязательный для заполнения реквизит.
  • Sent to warehouse (дата) - дата отправки на склад информации для отгрузки.
    Реквизит не редактирвется, а заполняется автоматически при нажатии на кнопку "Sent to warehouse".
    Не может быть позже "Sent to Move-it".
  • Sent to move it (дата) - дата отправки данных в систему "Move it".
    Реквизит не редактируется, а заполняется автоматически при нажатии на кнопку "Sent to move it".
  • Gross weight, kg. (число) - общий вес всех парселей, заполняется системой автоматически.
  • Сubic meters (число) - общий объем всех парселей, заполняется системой автоматически.
  • Created by (справочник Users) - пользователь, создавший документ, заполняется системой автоматически.
  • Creation date (дата и время) - время создания документа, заполняется системой автоматически.
  • Modified by (справочник Users) - пользователь, внесший последние изменения в документ, заполняется системой автоматически.
  • Modification date (дата и время) - время последнего изменения документа, заполняется системой автоматически.

Таблица "Parcels"

В таблице "Parcels" перечисляются грузовые места, которые относятся к текущему документу.

  • Parcel (справочник Parcels)
    Обязательный для заполнения реквизит (система автоматически удаляет пустые строки).
    Грузовые места не должны повторяться (система автоматически удаляет дубли).
    Грузовые места не должны быть указаны в других Trips.
    В грузовых местах не должно быть признака Repacked, должна стоять галочка Final.
    Warehouse from в грузовых местах должен соответствовать одноименному реквизиту шапки.
    При ручном выборе система автоматически накладывает необходимые фильтры.
  • Справочно отображаются следующие реквизиты: length, cm., width, cm., height, cm., gross weight, kg., cubic meters, warehouse to, PO list, hazard class, description.
    Кроме того в итогах таблицы показывается общий вес и объем.

При добавлении строки в таблицу открывается список грузовых мест, которые еще не вошли ни в один документ "Trip", но которые уже проверены и включены в Shipment на этап "Delivered to warehouse".
Двойное клик по грузовому месту открывает форму этого грузового места.
Кнопки изменения грузового места нет: для изменения необходимо удалить старое грузовое место и добавить новое.

Если заполнены Sent to warehouse и Sent to move it, список Parcels не должен быть пустым. Импортные parcels должны быть collected from port.

Таблица "Final destinations"

В таблице "Final destinations" отображается список всех конечных точек маршрута, а также указываются планируемые даты прибытия.

Пользователь не может добавить или удалить строку из этой таблицы, так как система автоматически определяет список конечных точек маршрута при изменении состава грузовых мест.

  • Warehouse to (справочник Destination points)
    Реквизит не редактируется, а определяется автоматически из грузовых мест.
    Значение Warehouse to не должно совпадать со значением Warehouse from.
  • ETA (дата) - estimated time of arrival
    Обязательный для заполнения реквизит.
    Не может быть раньше ATD.
