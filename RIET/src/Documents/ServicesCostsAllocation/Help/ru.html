Документ "Services costs allocation"

Документ "Services costs allocation" входит в подсистему "Services".
Предназначен для распределения стоимости сервисов на товары.

Нельзя изменить документ, если он относится к проведенному документу Agent invoice.

Реквизиты шапки документа

  • No. (строка) - автоматически присваиваемый номер документа.
  • Date (дата)
  • For deletion (булево) - признак того, что документ помечен на удаление.
  • Posted (булево) - признак того, что документ проведен.
    Нельзя провести помеченный на удаление документ.
  • Agent (справочник Agents)
    Обязательный для заполнения реквизит.
  • Comment (строка)
  • Creation date (дата) - дата создания документа.
  • Last modified (дата и время) - заполняется автоматически датой последнего интерактивного изменения документа.
  • Responsible (справочник Users) - заполняется автоматически пользователем, который внес последние изменения в документ

Табличная часть "Services"

  • Service (документ Service)
    Система автоматически удаляет пустые или повторяющиеся строки.
    Должен быть указан хотя бы один сервис.
    Сервис должен быть проведен.
    Сервис не должен входит в другие Services costs allocations.
    Сервис должен быть оказан агентом, указанным в шапке документа.
    Поставка, к которой относится данный сервис, должна быть закрыта документом Invoice lines matching.
    Есть возможность автоматического заполнения табличной части всеми не распределенными сервисами.
    При ручном выборе система автоматически накладывает необходимые фильтры.

Проведение

При проведении система распределяет стоимость услуг по товарам в регистр "Invoice lines costs" в соответствии с правилами заданными в услуге.

Одновременно происходит увеличения сумм в регистре "Not received agent invoices". Далее агент должен предоставить счета на эти суммы.
