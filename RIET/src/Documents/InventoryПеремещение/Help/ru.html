Документ "Inventory перемещение"

Документ "Inventory перемещение" предназначен для отражения факта перемещения inventory-товаров между inventory-локациями.

Реквизиты документа

Документ имеет реквизиты шапки и табличную часть "Goods".

Реквизиты шапки документа

  • Inventory location out (справочник Inventory locations) - локация-отправитель
    • Реквизит должен быть заполнен
  • Inventory location in (справочник Inventory locations) - локация-получатель
    • Реквизит должен быть заполнен
  • Segment (справочник Segments) - реквизит, определяемый автоматически из проводок DSS
  • Ict doc type (строка) - вид операции - справочный реквизит, подгружаемый из lawson
  • Has finished goods (булево) - автоматически устанавливаемый признак того, что в документе содержатся Finished goods
  • Creation date (дата и время) - автоматически заполняемая дата создания документа
  • Last modified (дата и время) - автоматически заполнеяемая дата последнего изменения документа из формы
  • Responsible (справочник Users) - пользователь, внесший последние изменения в документ
  • Comment (строка) - любой произвольный комментарий

Табличная часть "Goods"

В табличной части "Goods" перечисляются все перемещаемые товары, их количество и сумма.

  • Good (справочник Inventory goods) - товар
    • Реквизит должен быть заполнен
    • Good не может быть помечен на удаление
    • У Good должен быть заполнен реквизит Standard UOM
  • UOM (справочник UOMs) - единица измерения
    • Реквизит должен быть заполнен
    • UOM не может быть помечен на удаление
    • UOM standard UOM должен соответствовать Good standard UOM
  • Qty out (число) - количество, которое ушло с локации-отправителя
    • Реквизит должен быть заполнен
  • Qty in (число) - количество, которое пришло на локацию-получателя
    • Реквизит должен быть заполнен
  • Finished good (булево) - признак того, что для данного товара устанавливается ручной порядок списания
  • Finished good PO num. (строка) - номер PO, по которой должно происходить списание
    • Реквизит должен быть заполнен в том случае, если установлен признак Finished good
  • Management price out (число) - управленческая цена, по которой ушел товар с локации-отправителя - указывается справочно
  • Management price in (число) - управленческая цена, по которой пришел товар на локацию-получателя - указывается справочно
  • Management sum out (число) - управленческая сумма, по которой ушел товар с локации-отправителя - указывается справочно
  • Management sum in (число) - управленческая сумма, по которой пришл товар на локацию-получателя - указывается справочно

Проведение документа

При проведении документ перемещает партии товаров в регистре накопления "Inventory партии":

  • Inventory location - списывается с Inventory location out и приходуется на Inventory location in
  • Good - из строки документа
  • PO number - из строки документа 
  • Batch - подбирается автоматически по методу FIFO; при нехватке партии выдается ошибка
  • Quantity - списывается Qty out и приходуется Qty in; если Qty out превышает Qty in, то создается новая партия;
  • Sum - определяется из подобранной партии
