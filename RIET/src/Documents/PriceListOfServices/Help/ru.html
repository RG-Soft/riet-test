Документ "Price list of services"

Документ входит в подсистему "Services".
Предназначен для отражения цен услуг, оказываемых контрагентами при импорте / экспорте товаров.

Реквизиты шапки документа

  • No. (строка)
    Обязательный для заполнения реквизит.
    Контролируется уникальность.
  • Date (дата) - дата начала действия прайс-листа.
    Обязательный для заполнения реквизит.
  • For deletion (булево) - признак того, что прайс-лист помечен на удаление.
  • Posted (булево) - признак того, что прайс-лист проведен, а значит используется для определения цены указанных в нем услуг.
    Помеченный на удаление документ не может быть проведен.
  • Agent (справочник Agents)
    Обязательный для заполнения реквизит.
  • Expire date (дата) - дата окончания действия прайс-листа.
    Дата окончания не может быть раньше даты начала действия прайс-листа.
  • Description (строка) - любая информация, позволяющая идентифицировать прайс-лист.
  • Comment (строка)
  • Creation date (дата) - автоматически заполняемая дата создания прайс-листа.
  • Last modified (дата) - автоматически заполняемая дата последнего интерактивного изменения прайс-листа.
  • Responsible (справочник Пользователи) - пользователь, внесший последние изменения в документ.

Табличная часть "Standard tariffs"

  • Service (справочник Services)
    Обязательный для заполнения реквизит.
    Может быть указана только услуга с правилом формирования стоимости "Standard tariffs".
  • Description (строка) - наименование услуги, как оно будет фигурировать в актах.
    Обязательный для заполнения реквизит.
  • Price (число)
    Обязательный для заполнения реквизит.
  • Start date (дата) - дата начала действия строчки прайс-листа.
    Не может быть раньше даты начала действия прайс-листа.
  • Expire date (дата) - дата окончания действия строчки прайс-листа.
    Не может быть раньше Start date.
    Не может быть позже даты окончания действия прайс-листа.
