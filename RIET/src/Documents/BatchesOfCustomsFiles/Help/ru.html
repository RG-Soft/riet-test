Документ "Batches of customs files"

Документ "Batches of customs files" входит в одноименную подсистему.
Предназначен для объединения и выгрузки в Lawson документов Customs files и Customs files light (с типом транзакции Customs receipt order (ТПО), Customs bond (расписка) или Customs bond closing (закрытие расписки)).

Документ заполняется автоматически при нажатии на кнопку "Fill".
При этом система подбирает все документы Customs files и Customs files light, которые были закрыты до даты документа, но еще не выгружены другим документом Batch.

После проведения документ может быть распечатан для выгрузки в Lawson с помощью кнопки "Unload".

Кроме того, прямо из документа могут быть распечатаны отчеты "Batches of customs files (summary)", "Batches of customs files (details)" и "Покупная стоимость таможенных товаров".

Шапка документа

  • No. (строка) - автоматически присваиваемый номер документа.
  • Date (дата)
    Обязательный для заполнения реквизит.
  • For deletion (булево) - признак того, что данный документ помечен на удаление.
  • Sold-to (справочник Sold-to)
    Обязательный для заполнения реквизит.
    Sold-to не должен быть помечен на удаление.
  • Country
    Обязательный для заполнения реквизит.
  • Process level
    Обязательный для заполнения реквизит.
  • Comment (строка)
  • Created by (справочник Users) - заполняется автоматически пользователем, который первый создал документ.
  • Creation date (дата и время) - заполняется автоматически датой создания документа.
  • Modified by (справочник Users) - заполняется автоматически пользователем, внесшим последние имзенения в документ.
  • Modification date (дата и время) - заполняется автоматически датой последнего интерактивного изменения документа.

Табличная часть "Customs files"

В табличной части перечисляются документы Customs files of Import/Export shipments, включенные в текущий батч.

Табличная часть "Customs files of temp. imp. / exp."

В табличной части перечисляются документы Customs files, у которых в качестве shipment указан документ Temp. imp. / exp. transation на перевод товаров в постоянный режим, включенные в текущий батч.

Табличная часть "Customs receipt orders (ТПО)"

В табличной части перечисляются документы Customs files light с типом транзакции Customs receipt orders (ТПО), включенные в текущий батч.

Табличная часть "Customs bonds (таможенные расписки)"

В табличной части перечисляются документы Customs files light с типом транзакции Customs bond, включенные в текущий батч.

Табличная часть "Customs bond closings (закрытие таможенных расписок)"

В табличной части перечисляются документы Customs files light с типом транзакции Customs bond closing, включенные в текущий батч.

Для всех документов табличных частей проверяются следующие условия: 

  • Система автоматически удаляет пустые строки.
  • Система автоматически удаляет повторяющиеся строки.
  • Документ не должен быть указан в другом батче.
  • Sold-to в документе должен соответствовать Sold-to в шапке батча.
  • Дата документа Invoice lines matching для указанной декларации должна быть раньше или равна дате батча.
  • Документ должен быть проведен.
  • Invoice lines matching должен быть проведен.
  • Для соответствующей таможни должен быть прописан Vendor ID.
