Документ "Agent invoice"

Документ "Agent invoice" входит в подсистему "Services".
Предназначен для отражения в системе счетов поставщиков за оказанные услуги.

Реквизиты шапки документа

  • Agent no. (строка) - номер входящего документа.
    Обязательный для заполнения реквизит.
  • Fiscal no. (строка) - номер документа для целей фискального учета.
    Обязательный для заполнения реквизит.
  • Date (дата) - дата входящего документа.
    Обязательный для заполнения реквизит.
  • Received (дата) - дата получения счета.
    Обязательный для заполнения реквизит
  • For deletion (булево) - признак того, что документ помечен на удаление.
  • Posted (булево) - признак того, что документ проведен.
    Нельзя провести помеченный на удаление документ.
  • Agent (справочник Agents) - контрагент, который выставил счет.
    Обязательный для заполнения реквизит.
  • Sold to (справочник Sold to) - юр. лицо, на которое выставлен счет.
    Обязательный для заполнения реквизит.
  • Sum w/o VAT (число)
    Обязательный для заполнения реквизит.
  • VAT (число)
  • Sum with VAT (число)
    Обязательный для заполнения реквизит.
    Должен быть равен (Sum w/o VAT + VAT).
  • Comment (строка)
  • Creation date (дата) - автоматически заполняемая дата создания документа.
  • Last modified (дата и время) - заполняется автоматически датой последнего интерактивного изменения документа.
  • Responsible (справочник Users) - заполняется автоматически пользователем, который внес последние изменения в документ.

Табличная часть "Services"

В табличной части "Services" перечисляются все сервисы, оказанные в рамках данного счета.
Табличная часть может быть заполнена автоматически, при этом система добавляет все сервисы, по которым еще не получены счета.
Кроме того, есть возможность добавить сервисы из определенного документа Services costs allocation.

  • Service (документ Service)
    Система автоматически удаляет пустые и повторяющиеся строки.
    Должен быть указан хотя бы один сервис.
    Сервис не должен быть включен в другой счет.
    Сервис должен быть оказан контрагентом, указанным в шапке документа.
    При ручном выборе сервиса система автоматически накладывает необходимые отборы.
  • Sum (число) - реквизит должен быть заполнен
    Итог по табличной части должен совпадать со значением в реквизите Sum w/o VAT.
    При проведении контролируется то, что указанная сумма не превышает распределенную документом Services costs allocation.

Проведение

При проведении документ списывает указанные остатки из регистра "Not received agent invoices".
Далее данные можно выгрузить в Lawson, нажав на кнопку "Unload".
