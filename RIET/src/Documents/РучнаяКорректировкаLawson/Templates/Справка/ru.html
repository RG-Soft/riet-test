Загрузка из файла Excel

1. Загрузка осуществляется для первой станицы книги файла Excel. 

2. Первая строка страницы отведена под шапку, поэтому считывание данных производится со второй строки в соотвествие с указанным порядком колонок. 

3. Для ускорения загрузки из файла нужно обозначить окончание листа тремя звездочками (***) в первой колонке, т.е. программа дойдя до этой строки закончит считывание данных.
