Документ "Boxes of customs files"

Документ входит в подсистему "Customs documents".
Предназначен для учета коробок в которые кладутся оригиналы деклараций.

Реквизиты шапки

  • No. (строка) - автоматически присваемый порядковый номер.
  • Creation date (дата) - автоматически заполняемая дата создания документа.
  • Last modified (дата) - автоматически заполняемая дата последнего интерактивного изменения документа.
  • Responsible (справочник Users) - автоматически заполняемый пользователь, который последний внес интерактивные изменения в документ.
  • Dispatch date (дата)
    Обязательный для заполнения реквизит.
  • List of customs files (строка) - автоматически формируемый список номеров customs files для удобного поиска в списке документов
  • Comment (строка)

Таблица "Customs files"

В таблице перечисляются документы Customs files, которые входят в эту коробку.

Таблица "Customs files (light)"

В таблице перечисляются документы Customs files (light), которые входят в эту коробку.

  • Система автоматически удаляет пустые и повторяющиеся строки.
  • Должен быть хотя бы один Customs files или Customs files (light).
  • Документы должны быть проведены.
  • Документ не должена быть указан в другой коробке.
  • При выборе система автоматически накладывает необходимые фильтры.
