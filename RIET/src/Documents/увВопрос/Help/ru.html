Документ "Вопрос"

 

Документ фиксирует вопросы, задаваемые пользователями разработчикам. Для создания новых вопросов следует воспользоваться кнопкой "Добавить" в обработке "Вопросы". Документ могут создавать только те пользователи, которым дано соответствующее право в регистре сведений "Права пользователей".

При создании документа следует заполнить поля:

  • Вопрос — поле, содержащее текст вопроса,
  • Кому — элемент справочника "Пользователи", разработчик, которому задается Вопрос, или элемент справочника Группы пользователей, если на Вопрос может ответить любой пользователь из этой группы.
  • Тема вопроса — элемент справочника "Темы вопросов и заявок", содержит краткое описание сути Вопроса и используется для группировки Вопросов по темам. Служит признаком для отбора вопросов в списке,
  • Требуемая дата ответа — дата, к которой разработчику следует ответить на Вопрос,
  • Важность — важность Вопроса, может быть высокой, средней, низкой. По умолчанию ставится средняя важность.

Уже созданный Вопрос могут открывать следующие пользователи: задавший вопрос; тот, кому адресован вопрос; имеющий право просматривать все вопросы. 

 

Работа с Вопросом осуществляется следующим образом. Любой пользователь может добавить свое сообщение в документ, причем при добавлении определяется должен ли измениться статус всего вопроса и кто должен отвечать следующим, задавший вопрос пользователь или адресат исходного вопроса.

Статусы у Вопроса бывают такие: Новый, Утвержден, Принят к рассмотрению, Получен ответ, Закрыт. Статусы вопроса, а также связанные с ними даты изменения могут изменять следующие лица:

 

Утвержден — пользователи, у которых стоит право утверждать вопросы в регистре сведений "Права пользователей". Этот статус доступен только если в мастере настройки Монитора указана опция "Использовать утверждение вопросов".

Принят к рассмотрению, Получен ответ — пользователь-разработчик, которому задан Вопрос.

Закрыт — пользователь, задавший вопрос.

Для добавления сообщений в Вопрос следует пользоваться обработкой Вопросы или формой документа. Для определения вида сообщения и нового статуса Вопроса используются специальные кнопки командной панели (Уточнить вопрос, Принять к рассмотрению, Закрыть и т.п.).

В документ можно добавить произвольное количество Вложений. Вложениями могут быть ссылки на объекты базы данных, либо путь к файлу на диске. Для указания пути к файлу следует выбрать тип вложения Строка. При открытии вложения-файла будет запущена программа, ассоциированная в Windows с данным типом файлов. Следует понимать, что сам файл не сохраняется в базе данных, хранится только путь к файлу, поэтому следует указывать абсолютные пути к файлам, которые смогут открыть все пользователи, например, путь к файлу на сетевом диске.
