Документ "Blacklist"

Документ "Blacklist" входит в подсистему "Blacklist" и предназначен для поиска и отражения действий по товарам запрещенным или нежелательным к ввозу.

Новый документ может быть заполнен автоматически кнопкой "Fill" над табличной частью.
При этом система найдет все PO lines, которые удовлетворяют правилам из справочника "Rules of blacklist", но которые еще не указаны в инвойсах или других блеклистах.

Всем новым строкам автоматически присваивается статус "New", который по мере обработки может быть изменен на "In progress" или "Closed".
Кроме того, пользователь может указывать в каждой строке произвольный комментарий.

После этого документ следует провести, тогда он попадет в сводный отчет "Blacklist", в котором отображаются строки из всех блеклистов.

Структура

Реквизиты шапки

  • No. (строка) - автоматически присваеваемый уникальный номер документа.
  • Status (перечисление "Statuses of line of blacklist") - автоматически определяемый статус документа.
    Статус "New" присваивается, если все строки имеют статус "New".
    Статус "In progress" присваивается, если хотя бы одна строка имеет статус "In progress".
    Статус "Closed" присваивается, если все стркои имеют статус "Closed".
  • Posted (булево) - признак того, что документ проведен.
  • Creation date (дата) - автоматически присваиваемая дата создания документа.
  • Last modified (дата) - автоматически присваиваемая дата последнего изменения документа.
  • Responsible (справочник "Users") - автоматически присваиваемый пользователь, который внес последние изменения в документ.

Реквизиты табличной части "PO lines"

В табличной части должна быть введена хотя бы одна строка.

  • PO line (справочник "PO lines") - строка заявки на закупку.
    Обязательный для заполнения реквизит, если документ проведен.
    PO line не может быть помечена на удаление.
    Страна происхождения товара должна быть "Russian Federation".
    PO line не может встречаться в других не помеченных на удаление блеклистах.
    PO line не может повторяться несколько раз.
  • Rule (справочник "Rules of blacklist") - правило, по которому указанная PO line относится к запрещенным или нежелательным к ввозу.
    Обязательный для заполнения реквизит, если документ проведен.
    Rule не может быть помечено на удаление.
  • Status (перечисление "Statuses of line of blacklist") - текущий статус строки: New, In progress или Closed.
    Обязательный для заполнения реквизит, если документ проведен.
  • Comment (строка) - произвольный комментарий пользователя.
