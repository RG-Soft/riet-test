Siebel Orders (SO)

 

Каждый документ представляет собой аналог ордера из информационной базы Siebel. Все поля документа имеют строковый тип, соответствуют аналогичным из ордера и загружаются автоматически с помощью обработки «Загрузка ордеров из Siebel». 

 

Описание процесса загрузки::

  1. Открыть обработку через Invoicing - Загрузка данных из Siebel
  2. Указать имя файла без расширения, номер листа в файле Excel проставляется автоматически и равен 1 (нумерация листов начинается с 1).
  3. Нажать "Загрузить из Internet". В процессе загрузки все документы будут распределяться по трем закладкам: Новые, Измененные и Старые. Если SO с таким ID был загружен ранее и изменились какие-либо поля в шапке, то он попадает на закладку Изменен  ные, если не изменились, то на закладку Старые. Если такого ID  в базе не было – на закладку Новые.
  4. Проверить, правильно ли загрузились данные из файла. Не должно быть пустых дат документов!
  5. Нажать кнопку Создать документы. Будут созданы новые документы SO по каждой строчке на закладке Новые и изменены существующие для каждой строчки на закладке Измененные.
