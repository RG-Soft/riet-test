Обработка "Выгрузка данных"

Обработка осуществляет выгрузку данных подсистемы, а именно документов "Вопрос", "Заявка на выполнение работ", "Протокол тестирования", причем выгружаются сопутствующие данные справочников.

Для выгрузки данных необходимо установить период, за который следует производить выгрузку данных и файл выгрузки.

Существует также возможность отправить файл выгрузки на произвольный электронный почтовый адрес. Для этого необходимо указать один или несколько почтовых профилей в табличном поле "Отправка по email", по каждому профилю будет отправлено отдельное письмо. Взаимодействие с почтовым сервером осуществляется встроенным почтовым клиентом.
