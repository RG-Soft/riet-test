Обработка Pull master data from TMS

Обработка предназначена для получения данных из TMS:

  • контактные данные
  • данные локаций
  • данные номенклатуры
  • Order release by Order base

Обработка входит в подсистему TMS.

ВНИМАНИЕ!

WSDL файл с описанием веб-сервиса TMS был вручную исправлен и положен на наш веб-сервер сюда C:\inetpub\wwwroot
Дело в том, что TMS может присылать невалидные ответы, при этом 1ска падает.
Чтобы 1ска не падала мы вручную исправили WSDL файл, так как команда TMS исправить эту ошибку не смогла.
Список исправлений:

  • закомментирована строчка
    <xsd:element name="PERSPECTIVE" type="xsd:string"/>
    так как элемент Perspective уже определен выше, просто имеет строчные буквы, а 1с не различает этого.
  • закомментирована строчка
    <xsd:element name="DESCRIPTION" type="xsd:string"/>
    по той же причине
