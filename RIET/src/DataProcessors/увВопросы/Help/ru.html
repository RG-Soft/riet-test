Обработка "Вопросы "

Обработка является сновным интерфейсом пользователя для работы с подсистемой, она открывается из меню "Монитор сопровождения"/"Вопросы". В ней представлены вопросы, заданные пользователем,  адресованные ему или группе, в которую он входит, а также все остальные вопросы, если пользователь имеет право просмотра чужих вопросов;

Вопросы сгруппированы по статусам:

  • новые
  • утвержденные (если в настройках Монитора включено утверждение вопросов) 
  • принятые к рассмотрению
  • отвеченные 
  • к исполнению (Вопросы, адресованные пользователю, причем его черед отвечать)
  • все вопросы

Также можно установить дополнительный отбор только по своим вопросам и вопросам, адресованным групрам, в которые входит пользователь (галка Только мои и моих групп).

 

Основные реквизиты документов отображаются в табличном поле, тексты сообщений помещаются в правой части формы. Документы в списке можно сортировать и устанавливать для них отбор, эти действия совершаются стандартными способами.

 

При двойном нажатии на строке, представляющей документ, открывается форма документа, в которой можно его редактировать. При наличии соответствующих прав пользователь может выполнить действие над документом прямо из формы, для этого в нижней части формы размещены кнопки с поясняющими надписями. В командной панели имеется кнопка для ввода на основании документа «Заявка на выполнения работ».

 

Нажатием на кнопку "Печать текущего отчета" можно вывести печатную форму Вопроса. По нажатии на кнопку Заявки откроется список документов Заявка на выполнения работ». При нажатии на кнопку Поиск вопросов откроется обработка полнотекстового поиска по документам "Вопрос"

 

Через меню "Настройка..." можно открыть регистр для настройки прав пользователей подсистемы и мастер ввода параметров подсистемы "Монитор сопровождения".
