Обработка "Загрузка строк книги покупок из Excel"

Порядок работы.

Обработка находится в интерфейсе "PurchaseBook" в подменю "НДС" - пункт "Загрузка строк книги покупок из Excel".

Для загрузки необходимо произвести следующие шаги:

1. Указать файл-источник данных в формате "xls" или "xlsx".

2. Нажать кнопку "Загрузить" и дождаться пока табличное поле на форме будет заполнено данными указанного файла.

При загрузке строк книги покупок создаются документы "Счет книги покупок" с признаком "Загружен из Excel". Поэтому для корректной загрузки необходимо, чтобы были заполнены следующие поля: Дата проведения, Ваучер, Номер входящего документа, Дата входящего документа, Валюта, Курс, Продавец, Подразделение организации, СуммаДокумента с НДС, Сумма документа с НДС Руб, Сумма НДС, Сумма НДС Руб, а также Ставка НДС. В случае, если какое либо из перечисленных полей не удалось заполнить по данным указанного в обработке файла, то такая строка выделяется в табличном поле красным цветом, а также снимается признак "Создавать". Строки с не установленным флажком "Создавать" при загрузке игнорируются. Чтобы установить данный флажок вручную необходимо указать все поля, обязательные для заполнения.

Существуют также следующие особенности заполнения полей по данным файла: поле "Дата" заполняется из колонки "ActDate", а в случае отсутствия значения в данной колонке, из колонки "Date"; "Дата проведения" - максимальное из значения колонки "PostingDate", даты начала открытого периода по модулю Purchase Book и значения "Дата", определенного в данной строке.

3. Для окончательной загрузки данных необходимо нажать кнопку "Создать счета" и дождаться окончания выполнения операции.

4. Для закрытия обработки необходимо нажать кнопку "Закрыть" или соответствующую кнопку окна обработки.
