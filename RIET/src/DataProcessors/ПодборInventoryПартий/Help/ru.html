Обработка "Подбор Inventory партий"

Назначение

Обработка "Подбор Inventory партий" предназначена для автоматического формирования документов "Inventory партия".

Структура обработки

  • "Список документов Inventory партия" показывает сформированные за период документы "Inventory партия", сгруппированные по "PO Number"

Кнопка "Установить период" отбирает документы "Inventory партия" и позволяет автоматически создать новые документы за указанный период;

Кнопка "Создать партии за период" автоматически формирует документы "Inventory партия" за выбранный период;

"Дельта Fiscal sum выделенных элементов" суммирует разницу в фискальной стоимости IC и AP проводок по выделенным партиям/PO Number (доступно выделение нескольких элементов при удержании клавиши Ctrl);

Для списка доступен быстрый отбор непроведенных документов "Inventory партия".

  •  Табличная часть "Проводки деталей СКП по документу" является табличной частью документа "Inventory партия", выделенного в "Списке документов Inventory партия"

Для табличной части доступно только добавление/удаление проводок DSS. При удалении/добавлении проводкок DSS проведенный документ "Inventory партия" автоматически распроводится, сохраняя внесенные изменения.

Добавить проводку DSS можно с помощью перетаскивания проводки из "Остатков Inventory" в табличную или по кнопке "Добавить в партию", выделив нужные проводки.

  •  Список "Остатки Inventory" показывает все остатки IC и AP проводок по регистру Inventory

Для списка доступен быстрый отбор остатков по выделенному в "Списоке документов Inventory партия" документу или группе. По умолчанию ставится отбор "по всем", можно выбрать по значению "PO Number" или по значениям "PO Number" и "Item Code" (т.е. партии Inventory).

 

Segment

При выборе значения реквизита "Segment" будут отфильтрованы "Список документов Inventory партия" и "Остатки Inventory".

Проверки, выполняемые при добавлении Проводки DSS в  табличную часть "Проводки деталей СКП по документу"

- проверяется соответствие значений "PO Number" и "Item Code" в проводке значениям реквизитов документа "Inventory партия"

- проверяется наличие проводок DSS в других документах.

Проверки при записи/проведении документа "Inventory партия" указаны в справке к документу.
