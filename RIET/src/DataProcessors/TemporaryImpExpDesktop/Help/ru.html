Обработка 'Temporary import / export desktop'

Обработка предназначена для управления товарами, находящимися во временном импорте или экспорте.

Обработка входит в подсистему Temporary import / export.

Основные элементы

Обработка состоит из 3х частей:

  • Сверху располагаются кнопки управления обработкой
  • Ниже располагаются фильтры
  • Основное пространство занимает таблица товаров

Фильтры по-умолчанию

При открытии обработка автоматически устанавливает следующие фильтры:

  • Qty in temporary imp. / exp. > 0
    Система контроля временного режима устроена таким образом, что контролирует количество товаров, находящихся во временном импорте или экспорте.
    Например, по декларации во временный импорт принимается одна строка инвойса (товар), в которой указано 5 насосов.
    Сразу после этого qty in temporary imp. / exp. = 5.
    Со временем 2 насоса переводятся в свободное обращение.
    Это означает, что qty in temporary imp. / exp. = 3.
    Наконец, 3 оставшихся товара экспортируются.
    То есть Qty in temporary qty in temporary imp. / exp = 0.
    Таким образом, указанный выше фильтр показывает только те товары, которые сейчас находятся во временном импорте или экспорте.
  • Process level = Process level текущего пользователя.
    То есть система показывает только те товары, которая относятся к процесс-левелу текущего пользователя.

Пользователь может снять галочки с этих фильтров или добавить собственные фильтры.

Выделение цветом

В обработке предусмотрено выделение товаров цветом:

  • желтым выделяются товары, срок временного импорта / экспорта истекает в течение 3х месяцев.
    Список таких товаров каждый понедельник отправляется по электронной почте, указанной в process level.
  • красным выделяются товары, срок временного импорте / экспорта которых уже истек.

Управление состоянием товаров

Для изменения статуса товаров необходимо выделить один или несколько товаров и нажать на одну из кнопок сверху:

  • Create transaction with selected items
    Создает документ Temporary import / export transaction с выделенными товарами.
    При этом контролируется, что все выделенные товары относятся к одной декларации.
    Если необходимо вручную поставить товары на контроль временного режима, необходимо перейти в список документов Temporary import / export transaction и создать новый документ оттуда.
  • Change responsible of selected items
    Создает документ Temporary import / export transaction с видом операции Responsible change.
  • Show history
    Показывает список документов по выделенному товару.
    Двойной клик на документе открывает документ для просмотра и редактирования.
