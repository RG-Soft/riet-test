Настройки

Для работы обработки подключения обязательно нужно заполнить следующие поля:

  • Логин
  • Пароль
  • Адрес базы

 

Описание параметров обработки:

Обновлять список автоматически - список вопросов будет автоматически обновляться с интервалом указанным в поле Время авто. обновления

Загружать закрытые и отмененные вопросы - обработка будет при подключении к монитору загружать закрытые и отмененные вопросы(это может значительно увеличить сокрость загрузки списка вопросов, однако бывает полезно для просмотра истории обсужденных вопросов)

Только мои - в списке вопросов будут показаны только те вопросы, где присутствует в составе обсуждащих текущий пользователь

Показать полностью - определяет, будет ли показан текст вопроса целиком или часть содержания будет обрезана (для меньшего объема загружаемых данных)

Отображать контакты - на форме списка вопросов будет показана контактная информация по выделенному вопросу

Использовать помощник ввода вопросов - при вводе нового вопроса будет использоваться помощник, облегчающий на первых этапах ввод новых вопросов.
