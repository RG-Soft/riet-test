Порядок загрузки ОС из файла Excel.

1. Открыть обработку «Перемещение ОС по списку Excel».

2. В форме обработки следует указать:

          a. «Выберите файл с данными» - файл, из которого следует подгружать ОС.

          b. «Номер листа» - в файле Excel, из которого подгружаются ОС, может быть несколько листов, в этом поле следует указать порядковый номер листа в книге Excel (нумерация начинается с 1).

          c. «Начало» - номер строки в файле Excel, из которого подгружаются ОС, в которой находится информация о первом подгружаемом ОС.

          d. «Конец» - номер строки в файле Excel, из которого подгружаются ОС, в которой находится информация о последнем подгружаемом ОС.

          e. «Дата» - дата, на которую требуется создать документы перемещения.

          f. Номера колонок, описанные в форме обработки.

3. Нажать «Прочитать подразделения».

4. На закладке «Соответствие подразделений» указать подразделения базы.

4. Нажать «Переместить ОС»

5. После окончания загрузки в базе будут созданы документы «Перемещение ОС» за выбранную дату. Пользователю требуется провести созданные документы. При их проведении будут заполнены суммы в табличных частях документов.
