Обработка "Распределение материалов из книги покупок"

    Позволяет быстро создавать документы "Распределение материалов из книги покупок",  "Распределение Доп. расходов из книги покупок" и заполнять содержимое табличных частей простым методом drag&drop.

    Интерфейс обработки условно разделен на 2 части:

    1. Создать документы Распределение материалов из книги покупок и Распределение Доп. расходов из книги покупок;

    2. Вывести остатки для счетов книги покупок.

    Последовательность действий: сначала выводим остатки, затем создаем и заполняем документ.

    1. Нажимаем на кнопку "Обновить остатки" для заполнения поля остатков по книге покупок (Проводки DSS Деталей Счетов Книги Покупок (СКП)). Можно ограничить вывод остатков СКП, созданных за выбранный период.

    Проводки упорядочены по полю Posting Date по возрастанию. Если Сумма проводки указана в иностранной валюте, то Сумма руб будет расчитана по курсу на дату Posting Date.

     

    2. При выделении строки остатков СКП, поле PO заполняется списком номенклатуры из Purchase Order, указанному в проводке (PO Number). Двойным кликом по строке открываются в поле остатки - Проводка DSS, а в поле PO - Purchase Order. 

    По кнопке "Открыть документ" откроется документ Счет книги покупок, к которому относится выделенная строка остатков.

     

    3. Порядок заполнения нового документа:

    Поле со страницами FA, INVENTORY и M&S представляют собой табличные части документа Распределение материалов из книги покупок, поле Доп. расходы - табличную часть документа Распределение Доп. расходов из книги покупок.

    Для заполнения табличной части нового документа нужно перетащить строку из поля остатков СКП на соответствующую страницу FA, INVENTORY, M&S или Доп. расходы.

    Для предварительного (вспомогательного) определения к какому виду материалов относится Проводка СКП, для строк остатков с заполненным полем PO заполняется колонка Тип и строки остатков подсвечиваются следующими цветами:

    Для Тип = "FA", ЦветФона = Голубой

    Для Тип = "Inventory", ЦветФона = Зеленый

    Для Тип = "M&S", ЦветФона = Желтый

    Для Тип = "ДопРасходы", ЦветФона = Темно-розовый

    Для Тип = "Списать", ЦветФона = Сиреневый

    В поле Тип (над полем Остатков СКП) можно указать один или несколько Типов для отбора строк. 

    Заполнение даты документа

    При перетаскивании строки Дата создаваемого документа заполняется по следующему принципу: если Дата Периода меньше или равна Posting Date, тогда Дата создаваемого документа  = Posting Date и наоборот, если Дата Периода больше Posting Date, тогда Дата создаваемого документа  = Период. Дата может быть изменена вручную, но должна быть больше или равна периоду проводки, иначе будет выведено сообщение об ошибке.

     

    Заполнение номенклатуры в строках FA

    При добавлени строки, если не найдено соответствие по Item description в справочнике номенклатура, будет добавлен новый элемент справочника. Если в проводке не заполнен PO Number, будет выведено сообщение "Не найдено соответствие строке PO. Выберите вручную номенклатуру и заполните количество."

     

    Списание прочих расходов

    На страницу Списать должны попадать остатки, которые относятся к прочим (косвенным) расходам, не увеличивающим фискальную стоимость FA, INVENTORY, M&S. При нажатии на кнопку "Списать" для строк (проводок СКП) на странице Списать значение Fiscal type изменится на "AP##OTHER". Проводки с изменившимся значением Fiscal type не будут попадать в обработку распределения. 

     

    4. Остатки СКП можно переносить как в текущий создаваемый документ распределения, так и добавлять в ранее созданный документ. Для этого нужно нажать на гиперссылку "Выбрать документ распределения", выбрать из списка нужный документ для добавления остатков СКП (гиперссылка отобразит номер и дату выбранного документа). После чего заполнятся табличные части из выбранного документа или будет предложено очистить текущие строки табличных частей FA, INVENTORY, M&S или Доп. расходы.

    5. После заполнения табличных частей нажимаем на кнопку "Создать документ" для записи и проведения нового документа по регистрам.

     

    Дополнительно:

     

    При нажатии на желтую кнопку с крестиком, очищаются табличные части документа и выбранный по гиперссылке документ для распределения.

    В поле остатков СКП возможно выделение сразу нескольких строк для быстрого "перетаскивания" данных.

     

    По кнопке "История документов" можно посмотреть созданные в текущем сеансе работы документы Распределение материалов из книги покупок и Распределение Доп. расходов из книги покупок (последний созданный документ отображается в начале списка), а также Проводки СКП с измененным значением Fiscal type. После закрытия обработки История документов будет очищена.
