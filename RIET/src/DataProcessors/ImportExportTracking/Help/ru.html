Обработка "Import / export tracking"

Обработка входит в одноименную подсистему.
Обработка позволяет:

  • просматривать состояние импорта товаров
  • подписаться на обновления о состоянии импорт товаров по электронной почте
  • просматривать присоединенные файлы
  • создавать заявки на экспорт
  • просматривать состояние экспорт товаров

Вкладка "Import"

Вкладка Report предназначена для просмотра состояния импорта товаров.

В-начале необходимо указать по каким полям будет осуществляться поиск.
Дополнительно может потребоваться указать период запрашиваемых данных.
Далее кнопка Search выводит запрошенные данные.

Кнопка Documents and attachments позволяет открыть список присоединенных файлов.
Кнопка Save as .csv file позволяет выгрузить данные в файл формата csv.

Вкладка "E-mail substibtions"

Вкладка позволяет добавлять / удалять подписки на обновления о состоянии импорта товаров.

Система позволяет подписаться как на всю PO, так и на конкретный товар из этой PO.

Вкладка "Export requests"

Вкладка позволят создавать и просматривать состояние заявок на экспорт.
В настоящий момент эта функциональность находится в разработке.
