Обработка «Подключение к монитору сопровождения»

Обработка представляет собой интерфейс пользователя, содержащий список вопросов.

Для удобства поиска вопросов, к которым были добавлены новые сообщения и которые могут требовать ответа или других действий пользователя, применяется выделение строк цветом:

  • Вопросы, где пользователь является "Спрашивающим" и последнее сообщение добавлено «Отвечающим» выделяются светло-синим цветом;
  • Вопросы, где пользователь является "Отвечающим" и последнее сообщение добавлено пользователем «Спрашивающий» выделяются светло-красным.

Основные реквизиты вопросов отображаются в списке вопросов, тексты сообщений выделенных вопросов помещаются в правой части формы.

При двойном нажатии на строке, открывается окно, в которой можно добавить сообщение к вопросу.

Команды:

  • Задать вопрос - открывает форму нового вопроса
  • Завершить обсуждение - команда переводит вопрос в статус "Закрыт", данные вопросы по умолчанию не отображаются в списке вопросов, добавление новых сообщений в такие вопросы более невозможно. Закрыть вопрос может только пользователь, который является спрашивающим в вопросе.

Для организации оперативного подключения к компьютеру пользователя обработка (внизу слева) содержит гиперссылку для открытия программы AMMYY.
