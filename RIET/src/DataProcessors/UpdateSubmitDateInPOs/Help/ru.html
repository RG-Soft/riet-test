Обработка "PO loading"

Обработка "PO loading" входит в подсистему "PO" и предназначена для загрузки Purchase orders из csv-файла, сформированного системой SWPS.

Для загрузки необходимо выбрать csv-файл и нажать на кнопку Load.

Если при загрузке система обнаружит, что PO уже была загружена - она перейдет к следующей PO.
Однако, если пользователь отметил галочку "Replace existing POs", то данные PO будут обновлены.

Требования к файлу

Загрузка осуществляется из текстового файла, имеющего формат .csv.
В качестве разделителя полей должен использоваться символ ",".

Для успешного выполнения обработки необходимо, чтобы в файле присутствовали колонки со следующими названиями:

  • Borg Name
  • Purchaser name
  • ReqName
  • Submitter Name
  • PO Num
  • PO Create Date
  • ISUP
  • Supplier Name
  • Supplier Type
  • Supplier Country
  • Shipto Label
  • Line number
  • Commodity
  • Commodity Type
  • Type of Requisition
  • Supplier Part Num
  • Part description
  • Quantity
  • Unit Of Measure
  • Unit Cost
  • Currency
  • Extended Cost
  • Extended Cost (USD)
  • Ship Method
  • Shipping Instructions
  • Special Instructions
  • IP Activity Code
  • ERP Accounting Treatment
  • Cost center
  • ERP CC/AU

Порядок следования колонок не важен, система пытается определить расположение колонки по названию.
